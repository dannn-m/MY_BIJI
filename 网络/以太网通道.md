# 一、概念
## 1.
是一种lan间的交换机间技术，可以把多个快速以太网或者千兆以太网端口划分到一个逻辑通道中
![[Pasted image 20260415153007.png]]
# 2.优点
- 大多配置任务可以在一个通道接口完成（不是成员端口），可以确保链路中的配置一致
- 无需升级物理链路
- 负载均衡
- 提供冗余
# 3.限制
- 不能混用快速端口和千兆口
- 最多可以由8个配置兼容的端口组成，提供最高800mb/s或8gb/s（千兆口）
- 在两交换机间的成员端口配置需要相同（中继、本征vlan...)
- 在逻辑接口的配置应用于所有物理接口
## 4、自动协商协议实现
- 可以两个协议之一协商建立，端口聚合协议（PAgP）与链路控制聚合协议（LACP），协议允许交换机具有相似特征的端口协商为通道
**注意**：可以通过配置静态或者
