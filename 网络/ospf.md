## OSPF的组件
- 路由协议消息： ospf消息交换时会使用5种**数据包**来承载ospf路由消息
	- Hello数据包
	-  数据库描述数据包（BDB)
	- 链路状态请求数据包（LSQ）
	- 链路状态更新数据包（LSU）
	- 链路状态确认数据包（LSack）
- 数据结构：
	- 临接数据库-构建临接表
	- 链路状态数据库（LSDB）-构建拓扑表
	- 转发数据库-构建了路由表

| 数据库           | 表   | 说明                                                                                              |
| ------------- | --- | ----------------------------------------------------------------------------------------------- |
| 邻接数据库         | 邻居表 | - 列出了已经与本地路由器建立了双向通信的所有邻居路由器。<br>- 该表对于每个路由器都是唯一的。<br>- 查看命令：show ip ospf neighbor              |
| 链路状态数据库（LSDB） | 拓扑表 | - 列出网络中所有其他路由器的相关信息。- 代表了网络拓扑。<br>- 某个区域内的所有路由器都有相同的 LSDB。<br>- 查看命令：show ip ospf database      |
| 转发数据库         | 路由表 | - 列出了 SPF 算法根据链路状态计算生成的路由数据库。<br>- 每台路由器上的路由表都是唯一的，包含如何把数据包发送给其他路由器的信息。<br>- 查看命令：show ip route |
