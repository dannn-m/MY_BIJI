1.2 网络设计
1.2.1 设计说明
本设计方案严格基于标准的高可靠性“核心-汇聚-接入”三层分层架构进行设计，全面选用思科（Cisco）主流网络设备，确保方案在思科模拟器（Cisco Packet Tracer）中能够完美实现全功能仿真与配置验证。
1.
核心层设计说明：
核心层部署于行政楼1层网络中心核心机房。设计采用2台支持多层交换与硬件堆叠的高性能三层交换机 
Cisco Catalyst 3650-24PS。在模拟器中，通过启用三层路由功能（ip routing）作为全校的VLAN间路由中心（网关）与DHCP服务器。核心交换机向下通过万兆/千兆光纤链路聚合连接各分区汇聚交换机，向上连接出口路由器，保障全校骨干网的万兆线速转发及高冗余度。
2.
汇聚层设计说明：
按照校园地理分布与业务流量特性划分三个汇聚节点，均采用三层交换机 
Cisco Catalyst 3650-24PS：
西部汇聚： 负责收敛1#教学楼、2#教学楼及实验楼的日常教学与科研大数据流量。
中部汇聚： 部署于图书馆，独立收敛数字化阅览室及公共检索区的高密集吞吐流量。
东部汇聚： 部署于家属区入口弱电间，负责跨越 500米物理长距离 进行长距光纤数据收敛。
3.
接入层设计说明：
接入层深入各个楼宇及楼层，全面采用经典千兆下行、万兆/千兆上联的二层交换机 
Cisco Catalyst 2960-24TT-L 和 Cisco Catalyst 2960-48TT-L。接入层交换机直接面向PC终端、智慧多媒体讲台、安全监控摄像头及户内光网关等终端提供线速千兆接入，并通过开启端口安全及VLAN隔离技术提升边缘网络安全性。
4.
网络出口设计说明：
网络出口部署高性能多业务集成路由器 
Cisco ISR 4331（或在Packet Tracer中选用可自由扩展万兆单模光纤模块的 Router-PT），配置NAT（网络地址转换）、静态/动态路由协议（OSPF）以及ACL访问控制策略，提供10000M（10Gbps）高带宽外网接入，并向外安全交付Web及Email服务。
1.2.2 详细设计
1. 行政办公区（行政楼）
功能定位与布线： 行政楼包含校领导、财务、教务等核心管理机密部门，总计1463个有线点中的195个。核心机房设在1层，因此行政楼各层接入交换机直接双绞线上联至网络中心的核心交换机。
设备选型： 地下一层与一层共15个点，选用1台 WS-C2960-24TT-L；二层（65点）与三层（65点）密度高，各层分别选用2台 WS-C2960-48TT-L 进行级联或双联上联；四层（30点）与五层（20点）各选用1台 WS-C2960-48TT-L 与 WS-C2960-24TT-L。
2. 西部教学科研区（1#/2#教学楼、实验楼）
功能定位与布线： 1#与2#教学楼每层20点（各60点），全部为智慧教室；实验楼共5层，每层120点（共600点），属于典型的高密大数据流量区域。该区域通过西部汇聚交换机进行本地集中收敛。
设备选型： * 1#与2#教学楼：每层部署1台 WS-C2960-24TT-L，通过千兆双绞线或光纤汇聚至西部汇聚。
实验楼：每层高密度部署3台 WS-C2960-48TT-L（共15台），每层交换机通过千兆上行链路接入西部汇聚交换机。
3. 中部公共服务区（图书馆）
功能定位与布线： 共3层，每层基础信息点120个，2层另设1间拥有80台PC的电子阅览室，总计440个信息点。设立中部（图书馆）汇聚交换机。
设备选型： 一层与三层各部署3台 WS-C2960-48TT-L；二层由于叠置了电子阅览室（共200点），部署5台 WS-C2960-48TT-L。全馆接入交换机统一收敛至图书馆汇聚机柜。
4. 东部生活家属区（1#、2#、3#家属楼）
长距离传输攻克（500米空间跨度）： 网络中心（行政楼1F）至东部家属区的物理距离达500米，远超双绞线100米的电气物理限制。
模拟器物理实现方法： 在Cisco Packet Tracer中，核心交换机（Catalyst 3650）与东部汇聚交换机（Catalyst 3650）之间，必须选用“Fiber（光纤）”黄色线缆进行物理互联。
接口配置： 在交换机上插接标准的万兆单模光模块（在模拟器中直接连接交换机的 GigabitEthernet 或 TenGigabitEthernet 模块化光口），通过500米单模光纤提供无衰减、零丢包的万兆主干互联。
设备选型： 每栋家属楼（36户）各部署1台 WS-C2960-48TT-L 作为楼栋总控接入，通过光纤上联至东部汇聚交换机。

6. Cisco Packet Tracer 模拟器具体连线与接口规范：
外网出口段： 出口路由器 ISR 4331 的 Gi0/0/0 接口连接外网Internet，Gi0/0/1 接口通过交叉线/直通线连接核心交换机 C3650-A 的 Gi1/0/24（配置为三层接口 no switchport 并配IP）。
核心与服务器： 核心交换机的 Gi1/0/21-22 端口连接校内Web与Email服务器群。
骨干万兆/千兆主干（汇聚层上联）：
核心交换机 Gi1/0/1  西部汇聚交换机 Gi1/0/1（使用直通线/光纤）。
核心交换机 Gi1/0/2  中部（图书馆）汇聚交换机 Gi1/0/1。
核心交换机 Gi1/0/3  东部（家属区）汇聚交换机 Gi1/0/1（注意：模拟器中此段代表500米长距单模光纤连接）。
接入层下联： 各汇聚交换机的 Gi1/0/4-20 端口分别连接对应楼宇接入交换机（如 C2960）的 Gi0/1 或 Gi0/2 端口，下行端口 FastEthernet 0/1-48 直接连至终端PC。
